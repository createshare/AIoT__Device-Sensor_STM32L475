# AIoT 设备端

基于 STM32L475（STM32L475VET6 ） MCU 的 AIoT 设备端的平台搭建。



## STM32L475VET6

STM32L475VET6 是 ST（意法半导体）公司推出的一款 MCU（微控制器）。

STM32L475VET6 基本特性：

- ARM Cortex-M4 内核

- 最高主频为 80 MHz

- 512KB 片上 Flash

- 128KB 片上 SRAM

- 低功耗

- 100 个引脚，LQFP100封装

  

## IoT Board 物联网开发

IoT Board 物联网开发采用了 STM32L475VET6 作为主控芯片。

开发板上集成 ST-Link V2.1 仿真器、固件烧写器，无需购买任何仿真器。
